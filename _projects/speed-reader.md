## Overview

A Python tool that reads PDF files and prints out each word individually in a format designed to maximize reading speed. It’s intended to help users focus and process text faster by reducing eye movement.

## Goals

- Parse PDF files efficiently
- Display words in a controlled sequence for speed reading
- Provide a simple, terminal-based interface for quick reading sessions

## Tools & Skills Used

- **Python** — core logic and word display
- **PDF parsing** — extract text from PDF files
- **Terminal output** — sequential word presentation
- **Productivity tool design** — improving reading efficiency

## What I Learned

This project taught me how to handle text extraction and timed output in Python. I also explored ways to design tools that improve productivity and focus, learning that small interface choices (like word pacing and formatting) significantly affect usability.
