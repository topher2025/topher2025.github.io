## Overview

A command-line implementation of Minesweeper featuring full gameplay mechanics, including recursive cell reveals and game state management.

## Goals

- Recreate classic Minesweeper in a terminal environment
- Implement flood fill for empty cells
- Track win/loss conditions accurately

## Tools & Skills Used

- **Python** — game logic
- **2D arrays** — board representation
- **Recursion** — flood fill algorithm
- **Input handling** — user interaction

## What I Learned

This project strengthened my understanding of recursion and state management. It also showed how much logic is required behind even simple-looking games.
