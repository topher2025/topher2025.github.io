## Overview

A modular Python toolkit designed to assist CyberPatriot competitors with system hardening, auditing, and security checks. The project automates common tasks while keeping all actions transparent, safe, and competition-compliant.

## Goals

- Automate repetitive security checks and system hardening
- Ensure all actions are safe and non-destructive
- Provide clear logs and modular functionality for selective execution

## Tools & Skills Used

- **Python** — automation scripts and CLI interface
- **Linux administration** — auditing users, services, permissions
- **Security practices** — enforcing best practices and compliance
- **Modular design** — allow selective execution of tasks

## What I Learned

I learned how to design tools that are both safe and powerful, balancing automation with competition rules. The project strengthened my understanding of Linux system administration, security auditing, and Python-based automation frameworks. I also gained experience creating modular, maintainable, and user-friendly CLI tools.
